# Implementar Hortas Inteligentes com Tecnologia para Combater a Fome em Comunidades Vulneráveis

## Descrição

Projeto desenvolvido para a disciplina Atividades Extensionistas II - Tecnologia Aplicada à Inclusão Digital da UNINTER.

## Objetivo

Promover segurança alimentar e inclusão digital por meio da implementação de hortas inteligentes em escolas públicas e comunidades vulneráveis de Florianópolis - SC.

## Tecnologias Utilizadas

- Python
- Arduino
- Flutter
- MySQL
